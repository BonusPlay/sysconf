# NixOS configuration

### structure

```
files     - static files
home      - home manager configuration
pkgs      - wip nixos packages not yet merged to nixpkgs
machines  - nixos configurations per machine
modules   - unmerged nixos modules
secrets   - agenix encrypted secrets
```
